# 百度百科：人物类词条模板

> 本模板用于创建百度百科人物类词条，适用于企业家、创始人、行业专家等。
> 来源：百度百科官方人物类模板 2026

---

## 模板结构

### 提交前准备（隐私信息，不对外展示）

| 字段 | 说明 | 注意事项 |
|------|------|---------|
| 姓名（身份证名字） | 必须与身份证一致 | 用于实名认证 |
| 别名 | 网名、艺名、常用称谓 | 如：桥博士、马老师 |
| 手机号 | 联系方式 | 不对外展示，仅用于审核 |
| 身份证号 | 实名认证 | 不对外展示，仅用于审核 |

**重要提示**：
- 手机号和身份证号仅用于百度百科实名认证，不会在词条中公开显示
- 出生日期必须与身份证日期一致
- 准备1-20张人物照片（含一张封面图）

---

### 摘要（必填）

```
马某某，xxx年生于xx省xx市，xx公司创始人。现任xx公司董事长。
```

**填写要点**：
- 一句话说清：姓名、出生年份、出生地、主要身份、现任职务
- 字数控制在 50-100 字
- 语气客观中立，避免过度美化
- 突出最核心的身份标签

---

### 信息卡片（必填）

| 字段 | 说明 | 示例 |
|------|------|------|
| 中文名 | 真实姓名 | 乔烨 |
| 国籍 | 国籍信息 | 中国 |
| 民族 | 民族信息 | 汉族 |
| 籍贯 | 祖籍或出生地 | 江苏省南京市 |
| 出生日期 | 必须与身份证一致 | 1985年 |
| 毕业院校 | 最高学历院校（可不公布） | 同济大学 |
| 职务 | 当前主要职务 | XX公司董事长 |
| 就职企业 | 当前就职的企业 | XX科技有限公司 |

**注意事项**：
- 出生日期必须与身份证日期一致，否则审核不通过
- 毕业院校可以选择不公布
- 职务和就职企业填写当前最主要的身份

### 正文目录结构

#### 1. 主要经历（必填）

**格式**：时间轴形式，按时间顺序列出职业发展轨迹

```
20xx年，马某某担任xx公司商业运营体系副总裁。
20xx年x月，出任xx公司销售副总裁，负责xx方向。
20xx年x月，出任xx公司董事会主席兼CEO。
```

**填写要点**：
- 每条经历包含：时间 + 人物 + 职务/事件
- 按时间顺序排列（从早到晚）
- 选择职业生涯中的关键节点
- 时间精确到年份或月份
- 避免过度细节，保留重要转折点

**经历类型**：
- 教育经历：毕业院校、学位、专业
- 工作经历：任职公司、职务、主要职责
- 创业经历：创立公司、担任职务
- 重要项目：主导的重大项目或成果

#### 2. 社会职务（选填）

**格式**：列举当前担任的社会职务

```
xx公司董事长
xxx大学外聘讲师
xxx智库客座教授
xxxx研究院研究员
```

**填写要点**：
- 列举当前正在担任的职务
- 包括：企业职务、学术职务、行业协会职务、政府顾问等
- 按重要性排序
- 避免已卸任的职务

**职务类型**：
- 企业职务：董事长、CEO、总经理等
- 学术职务：客座教授、外聘讲师、研究员等
- 行业职务：协会会长、理事、专家委员等
- 政府职务：顾问、智库成员等

#### 3. 社会活动（选填）

**格式**：时间 + 人物 + 参与的活动

```
20xx年，马某某出席第二届经济论坛 
20xx年x月，主办xxx行业沙龙
20xx年x月，携公司旗下xx品牌亮相中国陶瓷博览会
```

**填写要点**：
- 列举参与的重要社会活动
- 包括：论坛演讲、行业会议、展会、公益活动等
- 选择有影响力的活动
- 时间精确到年份或月份

**活动类型**：
- 行业论坛：作为嘉宾演讲或参与讨论
- 展会活动：参展、主办、赞助
- 公益活动：慈善捐赠、志愿服务
- 媒体活动：接受采访、发表观点

#### 4. 所获荣誉（选填，需提供证书）

**格式**：荣誉名称 + 颁发机构

```
入选《中国企业家》最具影响力的企业领袖
荣获内蒙古建筑协会优秀企业家称号
荣获海南万宁市优秀共产党员
```

**填写要点**：
- **每一项荣誉都必须提供荣誉证书**
- 只列举权威机构颁发的荣誉
- 避免自设奖项或内部评选
- 按重要性或时间排序

**有效的荣誉类型**：
- 政府颁发：优秀企业家、劳动模范、先进个人等
- 行业协会：行业领军人物、杰出贡献奖等
- 权威媒体：年度人物、影响力榜单等
- 国际认证：国际奖项、全球榜单等

**无效的荣誉类型**：
- ❌ 企业内部评选
- ❌ 无法查证的奖项
- ❌ 自媒体评选
- ❌ 无证书的荣誉

#### 5. 人物相册（必填）

**要求**：
- 准备 1-20 张人物照片
- 其中必须包含一张封面图
- 照片要求清晰、正式
- 避免过度美化或修图

**照片类型**：
- 正式证件照或职业照（用作封面）
- 工作场景照片
- 演讲或活动照片
- 获奖照片
- 团队合影

## 百科审核要点

### ✅ 容易通过的内容

1. **可验证的经历**：
   - 在知名企业的任职经历（可通过企业官网、工商信息查证）
   - 在高校的学习经历（可通过学信网查证）
   - 公开的社会职务（可通过机构官网查证）

2. **权威荣誉**：
   - 政府颁发的荣誉（有证书）
   - 行业协会认证（有证书）
   - 权威媒体报道的榜单

3. **公开活动**：
   - 有媒体报道的论坛演讲
   - 有官方记录的展会参与
   - 有新闻稿的公益活动

4. **中立语气**：
   - 使用"担任"、"出任"、"参与"等客观动词
   - 避免"杰出"、"卓越"、"优秀"等主观评价

### ❌ 容易被拒的内容

1. **无法验证的信息**：
   - 无法查证的工作经历
   - 无证书的荣誉
   - 无媒体报道的活动

2. **过度美化**：
   - "著名企业家"、"行业领军人物"等自我标榜
   - "卓越贡献"、"杰出成就"等主观评价
   - 过度渲染个人能力

3. **商业推广**：
   - 详细介绍公司产品和服务
   - 包含联系方式和购买链接
   - 过度强调商业成就

4. **隐私信息不全**：
   - 出生日期与身份证不一致
   - 缺少实名认证信息
   - 照片不符合要求

## 参考资料要求

### 有效的参考资料来源

1. **企业官网**：关于我们、团队介绍页面
2. **权威媒体**：人民网、新华网、央视网、主流财经媒体的人物报道
3. **政府网站**：政府公告、表彰名单
4. **行业协会**：协会官网的职务公示
5. **企业信息平台**：天眼查、企查查（工商信息中的法人、高管信息）
6. **学术平台**：高校官网、学术机构官网

### 无效的参考资料来源

1. ❌ 自媒体文章（微信公众号、个人博客）
2. ❌ 论坛帖子（知乎、贴吧、豆瓣）
3. ❌ 无法访问的链接
4. ❌ 纯图片截图（需要文字链接）
5. ❌ 企业内部新闻稿（无第三方背书）

## 提交流程

### 第一步：准备材料
- [ ] 填写完整的信息卡片
- [ ] 整理主要经历（按时间顺序）
- [ ] 列出社会职务和社会活动
- [ ] 收集荣誉证书（每项荣誉都需要）
- [ ] 准备1-20张照片（含封面图）
- [ ] 准备参考资料链接

### 第二步：实名认证
- [ ] 提供真实姓名（与身份证一致）
- [ ] 提供手机号（不对外展示）
- [ ] 提供身份证号（不对外展示）
- [ ] 确认出生日期与身份证一致

### 第三步：提交审核
- [ ] 检查所有信息的准确性
- [ ] 确保语气客观中立
- [ ] 删除所有营销性内容
- [ ] 提交百度百科审核

### 第四步：补充完善
- [ ] 审核通过后，逐步补充内容
- [ ] 添加新的荣誉和活动
- [ ] 更新职务变动
- [ ] 保持信息准确性

---

## 注意事项

1. **实名认证是必须的**：百度百科人物类词条必须提供真实身份信息，否则无法通过审核
2. **出生日期必须一致**：词条中的出生日期必须与身份证上的日期完全一致
3. **荣誉必须有证书**：所有荣誉都需要提供证书或官方公示页面，否则不要写
4. **避免过度美化**：使用客观中立的语气，避免自我标榜
5. **先做内容铺垫**：在提交百科前，先在媒体平台发布人物报道，积累第三方来源
6. **分阶段提交**：先创建基础版本（只包含基本信息和主要经历），通过后再逐步完善
