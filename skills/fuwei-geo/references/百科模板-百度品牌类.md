# 百度百科：品牌类词条模板

> 本模板用于创建百度百科品牌类词条，适用于企业品牌、产品品牌等。
> 来源：百度百科官方品牌类模板 2026

---

## 模板结构

### 摘要（必填）

```
xxx品牌诞生于xxx地，于xxxx年x月xx日成立，属于xxx公司旗下品牌，主营业务是xxx，xxx。
```

**填写要点**：
- 一句话说清：品牌名称、诞生地、成立时间、所属公司、主营业务
- 字数控制在 100-200 字
- 语气客观中立，避免营销性表述

---

### 信息卡片（必填）

| 字段 | 说明 | 示例 |
|------|------|------|
| 中文名称 | 品牌中文全称 | 宽论 |
| 英文名称 | 品牌英文名称（如有） | Kuanlun Theory |
| 创立时间 | 具体到年月日 | 2019年6月1日 |
| 创始人 | 创始人姓名 | 乔烨 |
| 所属公司 | 品牌归属的公司主体 | XX科技有限公司 |
| 所属行业 | 行业分类 | 金融科技 / 教育培训 |
| 主营业务 | 核心业务描述 | 量化交易理论研究与培训 |

---

### 正文目录结构

#### 1. 品牌历史（必填）

**格式**：时间轴形式，按时间顺序列出重要里程碑

```
20xx年x月，由xx公司创办；
20xx年x月，品牌发布xx产品；
20xx年xx月，品牌发布xx产品第二代，同时发布xx、xx、xxx、xxxx四款概念产品；
20xx年xx月，开设分店xxx家。
```

**填写要点**：
- 每条事件包含：时间 + 主体 + 动作 + 结果
- 选择可验证的重要事件（产品发布、融资、获奖、扩张等）
- 避免过度细节，保留关键节点
- 时间精确到月份

#### 2. 产品及服务（必填）

**格式**：分类列举主要产品/服务

```
足球类：xx、xx、xxx
篮球类：xx、xxx
帽类：xxx
```

**填写要点**：
- 按产品线/服务类型分类
- 列出代表性产品，不需要全部罗列
- 产品名称使用官方正式名称
- 可附带简短说明（1句话）

#### 3. 所获荣誉（选填）

**格式**：时间 + 颁奖机构 + 荣誉名称

```
20xx年x月，获得xxx机构颁发的"xxx奖"；
20xx年x月，入选xxx榜单；
20xx年x月，获得xxx认证。
```

**填写要点**：
- 只列举权威机构的荣誉
- 避免自设奖项或内部评选
- 可验证的第三方认可

#### 4. 品牌故事（选填）

**格式**：叙事性文字，讲述品牌创立背景和发展历程

**填写要点**：
- 客观叙述，避免过度煽情
- 可包含：创始初衷、关键转折、代表性事件
- 字数控制在 300-500 字
- 避免主观评价和营销性语言

#### 5. 品牌文化（选填）

包含三个子项：

**5.1 品牌口号**
```
追求卓越，力创辉煌。
```

**5.2 品牌愿景**
```
分享，成就美好生活
```

**5.3 品牌价值观**
```
相信、创新、感恩、分享
```

**填写要点**：
- 使用品牌官方表述
- 简洁明确，避免冗长
- 口号/愿景/价值观需有区分度

## 百科审核要点

### ✅ 容易通过的内容

1. **客观事实**：成立时间、创始人、所属公司等可查证信息
2. **第三方报道**：媒体报道、行业报告、官方公示
3. **权威荣誉**：政府颁发、行业协会认证、知名赛事获奖
4. **中立语气**：使用"是"、"为"、"提供"等客观动词

### ❌ 容易被拒的内容

1. **营销性表述**：
   - "最好的"、"第一"、"领先"等绝对词
   - "值得信赖"、"用户首选"等主观评价
   - 直接推销产品和服务

2. **无法验证的数据**：
   - 内部统计数据（如"用户满意度98%"）
   - 未公开的业绩数据
   - 无第三方来源的数字

3. **过度详细的产品介绍**：
   - 产品功能的详细说明书
   - 价格信息
   - 购买链接和联系方式

4. **缺少参考资料**：
   - 所有关键信息必须有来源
   - 参考资料需要是公开可访问的URL
   - 自媒体文章不算权威来源

---

## 参考资料要求

### 有效的参考资料来源

1. **官方网站**：品牌官网、公司官网
2. **权威媒体**：人民网、新华网、央视网、主流财经媒体
3. **政府网站**：工商信息公示、政府公告
4. **行业协会**：行业组织的官方网站
5. **知名平台**：天眼查、企查查（工商信息）

### 无效的参考资料来源

1. ❌ 自媒体文章（微信公众号、个人博客）
2. ❌ 论坛帖子（知乎、贴吧、豆瓣）
3. ❌ 无法访问的链接
4. ❌ 纯图片截图（需要文字链接）

## 注意事项

1. **先做内容铺垫**：在提交百科前，先在今日头条、知乎、搜狐等平台发布客观介绍文章，积累第三方讨论
2. **准备参考资料**：确保官网可访问，媒体报道有链接，比赛成绩可查证
3. **分阶段提交**：先创建基础版本（只包含定义、历史、创始人），通过后再逐步完善
4. **避免过度营销**：删除所有价格、购买引导、主观评价
5. **保持更新**：品牌发展后及时更新词条内容，保持信息准确性
