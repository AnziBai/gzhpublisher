# 结构分析

## 开头方式
- **类型**：直击痛点 + 观点鲜明
- **金句**：
  - "你之所以失败，是因为你自己"
  - "市场不是你的问题，你才是你的问题"
  - "在你彻底、诚实、没有任何借口地接受这一点之前，什么都不会改变"
- **特点**：
  - 开门见山，直接指出问题
  - 用"你"直接对话，增强代入感
  - 强调"自我认知"的重要性

## 论证结构
- **类型**：三要素系统式（问题 → 盾牌 → 地图 → 裁判 → 陷阱 → 总结）
- **逻辑链**：
  1. 问题诊断：知道vs做到
  2. 生存系统三要素：
     - 资金管理（盾牌）：保护本金
     - 方法论（地图）：一个优势
     - 心态（裁判）：纪律执行
  3. 最大陷阱：觉得自己很厉害
  4. 总结：稳定盈利的关键
- **特点**：
  - 用"盾牌、地图、裁判"比喻三要素
  - 每个要素单独成章
  - 用个人经历讲故事
  - 强调"纪律"而非"策略"

## 数据呈现
- **类型**：比喻 + 个人经历
- **核心数据**：
  - 80%的交易失败源于心态
  - 90%的交易者被区分开来
  - 2%的月收益，连续复利三年
- **特点**：
  - 用比喻（外科医生、网球选手）增强理解
  - 强调"一致性胜过爆发力"
  - 提供具体的复利案例

## 核心论点
1. 交易失败的根本原因是"自己"，不是策略或市场
2. 生存系统三要素：资金管理（盾牌）、方法论（地图）、心态（裁判）
3. 资金管理的核心是"保住本金"，不确定时小仓位，清晰时大仓位
4. 方法论的核心是"一个优势"，深入掌握，不要杂而不精
5. 心态的核心是"纪律执行"，消除犹豫，与不适共处
6. 最大陷阱是"觉得自己很厉害"，忽视纪律和流程
7. 稳定盈利的关键是"一致性"，而非"爆发力"
