# 结构分析

## 开头方式
- **类型**：定义引入 + 历史背景
- **金句**：
  - "艾略特波浪理论是一种技术分析理论，用于预测未来价格趋势"
  - "波浪理论是由美国经济哲学家拉尔夫•艾略特创立，1938年他在研究历史股价走势变化"
- **特点**：
  - 开门见山给出定义
  - 强调理论的历史渊源和创始人背景
  - 用"海浪"比喻解释波浪概念

## 论证结构
- **类型**：教学式（定义 → 基本形态 → 原则 → 规则 → 应用方法 → 结合工具）
- **逻辑链**：
  1. 定义波浪理论（是什么）
  2. 基本形态（5波推动3波调整）
  3. 三个原则（推动波规则）
  4. 多空行情规则（三铁律）
  5. 三种交易方法
  6. 结合斐波那契回调线
  7. 总结
- **特点**：
  - 层层递进，从理论到实践
  - 大量使用图示说明
  - 提供具体的交易方法
  - 强调规则和原则

## 数据呈现
- **类型**：图示为主 + 规则说明
- **核心数据**：
  - 5波推动 + 3波调整 = 8波循环
  - 三个原则（第3波不能最短、第2波不超起点、第4波不与第1波重叠）
  - 多空三铁律
  - 斐波那契回调比例（23.6%、38.2%、50.0%、61.8%）
- **特点**：
  - 用图示直观展示波浪形态
  - 用实例（美元/日元）验证理论
  - 强调规则的重要性

## 金句提取
- "波浪理论更像是观察市场透露出的现象，经由验证去找寻、定义市场规律"
- "横看成岭侧成峰，远近高低各不同，不识庐山真面目，只缘身在此山中"
- "波浪理论说白了就是尝试发现这个波动，当你清楚自身位在哪个阶段，交易的节奏就不会打乱"
- "掌握波浪理论就跟掌握呼吸法一样，让交易者轻松应对市场状况"

## 结尾方式
- **类型**：哲学升华 + 价值强化
- **特点**：
  - 用苏轼诗句升华主题
  - 强调"发现"而非"发明"
  - 比喻波浪理论为"呼吸法"
  - 强调掌握节奏的重要性
