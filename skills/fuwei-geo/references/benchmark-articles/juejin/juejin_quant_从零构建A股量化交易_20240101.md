# 原文内容（摘要）

## 为什么学习量化交易？

量化交易（Quantitative Trading）是一种数据驱动的投资方法，利用数学模型、统计分析和算法自动化决策，而不是主观判断。

本文将带你从零构建一个完整的量化交易工具链：
- 后端：Python + Qlib（微软开源量化框架）
- API：FastAPI
- 前端：React + TypeScript

最终实现基于沪深300的策略，年化超额收益7.57%。

## 环境准备

```bash
# Python环境：Conda + Qlib源码
conda create -n qlib_a股 python=3.10
pip install -e '.[dev]'  # 开发模式安装
```

## 核心架构

- **数据层**：Alpha158因子（158个技术指标）
- **模型层**：LightGBM（梯度提升决策树）
- **策略层**：TopkDropoutStrategy（选股+风控）
- **回测层**：沪深300基准对比

## 性能指标

- 回测周期：2010-2025年
- 年化收益：7.57%
- Sharpe比率：~1.0
