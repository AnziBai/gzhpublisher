# 爆款文章数据库

## 目录结构

```
benchmark-articles/
├── README.md                    # 本文件
├── index.md                     # 文章索引（标题、来源、结构框架、金句）
├── wechat/                      # 微信公众号爆款文章
├── juejin/                      # 掘金平台爆款文章
├── toutiao/                     # 今日头条爆款文章
├── zhihu/                       # 知乎平台爆款文章
├── xueqiu/                      # 雪球平台爆款文章
└── jukuan/                      # 聚宽平台爆款文章
```

## 收录标准

**数据指标**：
- 点赞量 > 100
- 收藏量 > 50
- 阅读量 > 10,000
- 评论量 > 20

**内容标准**：
- 主题相关：缠论、波浪理论、量化交易、技术分析、散户投资
- 结构清晰：有明确的开头、论证、结尾
- 有金句：可提取的精彩表达
- 有数据：具体的案例或数据支撑

## 文件命名规范

格式：`{平台}_{关键词}_{标题前10字}_{日期}.md`

示例：
- `juejin_chanlun_缠论学了3年还是亏_20260421.md`
- `toutiao_quant_散户怎么入门量化_20260420.md`

## 文件内容格式

每篇文章包含：

```markdown
---
title: 原文标题
source: 平台名称
url: 原文链接
author: 作者名
publish_date: 发布日期
stats:
  views: 阅读量
  likes: 点赞量
  favorites: 收藏量
  comments: 评论量
keywords: [关键词1, 关键词2, 关键词3]
---
```

## 使用方法

1. **查找对标文章**：根据文章类型（对比类、战绩类、方法论类）查找对应的爆款文章
2. **提取结构框架**：分析开头、论证、结尾的方式
3. **学习写作手法**：提取金句、数据呈现、逻辑链
4. **一比一模仿**：保留结构，替换内容为宽论相关
